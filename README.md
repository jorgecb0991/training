# Catálogo de Teléfonos Móviles

Este proyecto es una aplicación de consola desarrollada en Java que permite a los usuarios explorar un catálogo de teléfonos móviles, seleccionar operadores (Claro, Entel, Movistar) y consultar diversos planes disponibles.

## Propósito
El objetivo principal es demostrar el uso de Programación Orientada a Objetos (POO) en Java, manejando colecciones de datos, clases POJO y lógica de selección por consola. Fue desarrollado originalmente como parte de un **Examen Parcial**.

## Tecnologías Utilizadas
- **Lenguaje**: Java SE
- **Herramientas**: Java Collections Framework, Scanner para entrada de datos.

## Guía de Uso
1. Compila los archivos fuente en el directorio `src`:
   ```bash
   javac src/*.java
   ```
2. Ejecuta la clase principal `Ejercicio1`:
   ```bash
   java -cp src Ejercicio1
   ```
3. Sigue las instrucciones en la consola para seleccionar un teléfono, operador y plan.
